Introduccion

La lógica difusa es una a la lógica clásica que pretende introducir un grado de vaguedad en las cosas que califica. En el mundo real existe mucho conocimiento no-perfecto o no-exacto, es decir impreciso, incierto, ambiguo, inexacto, o probabilístico por naturaleza.
El razonamiento y pensamiento humano frecuentemente conlleva información de este tipo, que quizá está originada de la inexactitud inherente de los conceptos humanos y del razonamiento basado en experiencias similares pero no idénticas a experiencias anteriores. El problema principal surge de la poca capacidad de expresión de la lógica clásica.
La solución que presenta la lógica clásica es definir un umbral de pertenencia donde dicho umbral define todo elemento perteneciente a un conjunto determinado, mientras que los otros no. Así cualquier elemento por mínima diferencia respecto a su siguiente conjunto será calificado como absoluto "A" o de "B", considerando dos conjuntos clásicos cualesquiera.
Sin embargo, si dispusiéramos de una herramienta para caracterizar por ejemplo, las alturas de varias personas de forma que las transiciones entre pertenencia a conjuntos fueran suaves, estaríamos reproduciendo la realidad mucho más fielmente.

function [YC] = Problema4(x,A)
TC=input("elija el tipo de complemento deseado 2=sugeno 3=estandar 1=yager");
y=A;
if TC==1
w=input("seleccione un valor para w mayor a 0");
C=(1-(y.^w)).^(1/w);
YC=C;
end
if TC==2
s=input("indique un valor de s mayor a -1");
C=(1-y)./(1+(s*y));
YC=C;
end
if TC==3
C=1-y;
YC=C;
end
subplot(2,1,1)
plot(x,y,"g");
title("conjunto original");
subplot(2,1,2);
plot(x,YC,"r");
title("complemento del conjunto")
end

function [YC] = Problema5(x,A,B)
TC=input("elija el tipo de intersección deseado 1=min 2=Producto algebraico 3=Producto acotado");
if TC==1
for i=1:length(A)
if A(i)>=B(i)
YC(i)=B(i);
else
YC(i)=A(i);
end
end
end
if TC==2
YC=A.*B;
end
if TC==3
y=A+B-1;
for i=1:length(A)
if y(i)>=0
YC(i)=y(i);
else
YC(i)=0;
end
end
end
subplot(3,1,1)
plot(x,A,"g");
title("conjunto original A");
subplot(3,1,2);
plot(x,B,"r");
title("conjunto original B")
subplot(3,1,3);
plot(x,YC,"b");

end
